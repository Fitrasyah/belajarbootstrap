Minat

HTML, CSS,dan JavaScript

HTML, CSS, dan JavaScript adalah tiga teknologi dasar yang digunakan untuk membuat website. HTML mendefinisikan struktur dan konten halaman web, CSS mengatur tampilan visual dan tata letaknya, sementara JavaScript menambahkan interaktivitas dan fungsionalitas dinamis.

Kompetensi

HTML dan PHP

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan tampilan dasar halaman web, sedangkan PHP (Hypertext Preprocessor) adalah bahasa pemrograman sisi server yang digunakan untuk menghasilkan konten dinamis dan interaktif pada halaman web. HTML berfungsi sebagai kerangka statis yang akan diolah oleh browser, sedangkan PHP membutuhkan web server untuk memproses kode dan mengirimkan hasilnya ke browser dalam bentuk HTML.
